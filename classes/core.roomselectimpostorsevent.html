Class RoomSelectImpostorsEvent<RoomType>

Emitted when the room picks which impostors will be selected, after the game is started. Can be canceled to avoid choosing and setting impostors altogether.

Mainly emitted in order to allow overriding of the default impostor selection in SkeldJS.

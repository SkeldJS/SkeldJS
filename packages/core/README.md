## @skeldjs/au-core

This package contains internal APIs for important structures in Among Us, and while you can install it on its own with `npm install --save @skeldjs/au-core` or `yarn add @skeldjs/au-core`, it is one package of a bigger project, [skeldjs](https://github.com/skeldjs/SkeldJS).

You can view auto-updating documentation for this package hosted at github pages at https://skeld.js.org/modules/core.html
